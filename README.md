# PUCIT CGPA & GPA Calculator

An interactive web-based GPA and CGPA calculator built by **Muhammad Abdullah** for PUCIT students.
It implements the official grading scale, rounding rules (ceiling), and CGPA policy described by PUCIT.

## Features
- **Semester GPA** calculation from course-wise marks and credit hours.
- **Cumulative CGPA (all courses)** directly from individual courses using credit hours as weights.
- **Updated CGPA** mode that combines a previous CGPA + credits with the current semester to get a new CGPA.
- Grade equivalence table (marks → letter grade → grade point).
- Clear explanation of CGPA formula and worked example.
- Mobile-friendly UI with dark theme and social links (LinkedIn & GitHub).

## Technology
- Pure **HTML**, **CSS**, and **JavaScript** (no backend required).

## How to use
1. Open `index.html` in your browser.
2. Choose a **Mode** from the header (Semester GPA, Cumulative CGPA, or Updated CGPA).
3. Enter each course with its **credit hours** and **obtained marks**.
4. Click **Calculate** to see GPA/CGPA and detailed breakdown.
